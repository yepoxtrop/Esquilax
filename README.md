# ESQUILAX 🐻

![Esquilax](./src//others//banner.png)

## Descripcion de la aplicacion 🤖
Esquilax es una aplicación de escritorio que permite descargar videos de YouTube de manera sencilla y rápida. Con una interfaz intuitiva y fácil de usar, Esquilax es la herramienta perfecta para aquellos que desean guardar sus videos favoritos para verlos sin conexión.

## Caracteristicas principales 🌟
- Descarga de videos de YouTube en alta calidad.
- Interfaz de usuario amigable e intuitiva.
- Soporte para múltiples formatos de video.
- Proceso de descarga rápido y eficiente.
- Actualizaciones regulares para mejorar la experiencia del usuario.
